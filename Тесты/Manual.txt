В каждой папке есть .cpp файл, используйщий файл binomial_heap.h. Это сделано для реализации разных интерфейсов для разных тестов.

Если в папке нет описания использования тестов, то использовать это:

На первой строке должно быть введено количество операций

Каждая операция должна начинаться с новой строки. 
Синтаксис операций:

- Создать кучу

CREATE HEAP

Примечание: создается новая куча с номером, полученным в порядке добавления. Нумерация куч начинается с нуля.

Для работы с кучами необходимо обращаться к ней по индексу, после чего указывать операцию
Примеры:

HEAP 3 INSERT 1
HEAP 2 MERGE 4
HEAP 5 EXTRACT_MIN




- Вставить элемент

INSERT [число]

Если необходимо хранить указатель на созданный объект, то необходимо использовать такую конструкцию

INSERT_STORABLE [число] 

После вставки указатель на новый элемент сохранятся и имеет номер, полученный в порядке добавления.
Нумерация указателей начинается с нуля.




- Извлечь минимум

EXTRACT_MIN

Примечание: указатель на извлеченный элемент становится недействительным



- Получить минимум

GET_MIN




- Удаление элемента

DELETE [номер указателя на элемент]

Примечание: указатель на удаленный элемент становится недействительным




- Объединение куч

MERGE [номер первой кучи] [номер второй кучи]

Примечание: в результате объединения куч создается новая куча, а предыдущие удаляются. Нельзя слить кучу саму с собой.




- Уменьшить значение элемента

DECREASE_KEY [номер указателя на элемент] [новое значение]




ПРИМЕЧАНИЕ: при попытке обращения к несуществующему элементу, извлечения минимума из пустой кучи, увеличения значения и т.д. программа завершает работу.
